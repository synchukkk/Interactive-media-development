├── index.html          # Головна сторінка
├── styles/
│   └── style.css       # Стилі
├── scripts/
│   └── main.js         # Ігрова логіка
└── README.md          # Опис проекту
🎮 Особливості гри

Система рівнів: Прогрес через досягнення цілей
