MindFlow demo

Інструкція:

1. Відкрийте файл `index.html` у браузері (подвійний клік або "Open File").
2. Компонент використовує React + Tailwind через CDN — інтернет-підключення обов'язкове.
3. Навігація між секціями: клік або клавіші ArrowLeft / ArrowRight, Home / End. Вибрана секція зберігається в localStorage.

Примітки:
- Цей демо-файл призначений для локальної перевірки. Для продакшну рекомендовано зібрати проект з нативним бандлером (Vite, Next.js тощо).
- Якщо бажаєте, можу конвертувати код у React-проект (create-react-app / Vite) або розбити на окремі файли компонента.
